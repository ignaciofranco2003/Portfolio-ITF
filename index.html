Sobre mí

Soy desarrollador Backend / Full Stack y estudiante de 5º año de la Licenciatura en Informática. Mi perfil técnico se centra principalmente en el ecosistema Java con Spring Boot para el desarrollo de sistemas backend robustos y escalables, y en Python con Flask para la creación de servicios y aplicaciones más ligeras y flexibles.

Tengo experiencia en el desarrollo de APIs REST, en el modelado y gestión de bases de datos MySQL, y en la implementación de mecanismos de autenticación y autorización utilizando roles, sesiones y JWT. Manejo herramientas del entorno profesional como Git y GitHub para control de versiones y Postman para pruebas y validación de servicios.

Además, cuento con formación en desarrollo web utilizando HTML, CSS y JavaScript, integrando estas tecnologías con sistemas backend. Mi objetivo es continuar desarrollándome profesionalmente en el área de backend y full stack, participando en proyectos donde pueda aportar valor técnico y seguir profundizando en arquitectura de software y diseño de sistemas.
